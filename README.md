# Weather REST API Client (Java)

This is a Java application that consumes a public REST API and displays weather data.

## Features

- Sends HTTP GET request
- Consumes REST API
- Parses JSON response
- Displays structured weather data

## Technologies used

- Java
- REST API
- JSON parsing (org.json)

## How to run

Compile:

javac -cp .;json-20231013.jar WeatherClient.java

Run:

java -cp .;json-20231013.jar WeatherClient
